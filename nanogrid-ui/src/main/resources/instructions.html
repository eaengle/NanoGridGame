How to Play Nano Grid

Nano Grid is a nonogram (picross) puzzle. Fill in cells on the grid so that the numbers on each row and column describe the filled groups in that line.

Controls

  • Left-click an empty cell to fill it (#).
  • Left-click a filled cell to mark it (X) — marking means "definitely empty here".
  • Left-click a marked cell to clear it.
  • Click and drag (left button) to fill or clear multiple cells in a row.
  • Right-click a cell to start a distance guide — move along the same row or column to see the span count in the corner.

Number Clues

  • Each number describes a consecutive run of filled cells in that row or column.
  • Multiple numbers mean multiple separate groups, in order, with at least one empty cell between them.
  • Example: "3 1" means a run of three filled cells, then a gap, then one filled cell.

Hints Menu

  • Check — counts how many filled cells you placed incorrectly.
  • Peek — briefly reveals the solution for one second.
  • Show — reveals the full solution.

File Menu

  • Save Game — saves your current puzzle and all filled cells.
  • Load Game — restores a saved game including your progress.
  • Save Puzzle — saves just the puzzle (no play state).
  • Load Puzzle — loads a puzzle and clears your progress.

Settings

  • Grid Options — change grid size and puzzle generation parameters.
  • Refresh Puzzle — clears all your filled cells, keeping the same puzzle.
  • New Puzzle — generates a brand-new random puzzle.
