How to Play Nano Grid

Nano Grid is a nonogram (picross) puzzle. Fill in cells on the grid so that the numbers on each row and column describe the filled groups in that line.

Controls

  • Cycle mode changes a cell from filled, to marked, to clear.
  • Fill, Mark, and Erase modes apply that state directly.
  • Left-click a cell to apply the selected mode.
  • Left-click and drag to apply the same state across multiple cells.
  • Right-click and drag along one row or column to show a span count in the corner.
  • Ctrl+Z / Ctrl+Y — Undo / Redo cell changes.
  • Ctrl+N — Open the New Puzzle dialog.
  • Ctrl+R — Refresh (clear all progress on the current puzzle).
  • Ctrl+K — Check how many filled cells are incorrect.
  • Ctrl+S / Ctrl+O — Save / Load game.
  • 1, 2, 3, 4 — Switch between Cycle, Fill, Mark, and Erase modes.

Number Clues

  • Each number describes a consecutive run of filled cells in that row or column.
  • Multiple numbers mean multiple separate groups, in order, with at least one empty cell between them.
  • Example: "3 1" means a run of three filled cells, then a gap, then one filled cell.

Side Buttons

  • Check (Ctrl+K) — counts how many filled cells you placed incorrectly.
  • Peek — briefly reveals the solution for one second.
  • Show — reveals the full solution.
  • Dark — toggles dark mode on the puzzle board.

Toolbar

  • New (Ctrl+N) — opens puzzle settings and generates a new puzzle.
  • Undo (Ctrl+Z) and Redo (Ctrl+Y) — step through recent cell changes.
  • Refresh (Ctrl+R) — clears your progress on the current puzzle.
  • Save (Ctrl+S) — saves the current game or puzzle to a file.
  • Load (Ctrl+O) — loads a saved game or a puzzle with empty progress.
  • ? — opens these instructions and the About dialog.

New Puzzle Options

  • Difficulty — controls how densely filled the puzzle is. Easy has more empty cells; Hard is more filled and harder to solve uniquely.
  • Symmetric — generates a puzzle with 180° rotational symmetry, which gives it a more balanced appearance.
  • Seed — enter a whole number to generate the same puzzle again later. Leave blank for a random puzzle each time.
  • Columns / Rows — set the grid size directly.
